# **App Name**: ShellShock Counter

## Core Features:

- Blank Shell Increment: Increments the blank shell counter and triggers a visual and auditory confirmation, including anime-style spark/particle animation.
- Active Shell Increment: Increments the active shell counter and triggers a visual and auditory confirmation, including anime-style spark/particle animation.
- Live Counter Display: Displays real-time counts for both blank and active shells with animated updates and great contrast for accessibility.
- Shell Count Reset: Allows users to reset shell counts via a modal prompt asking for the number of active and blank shells, followed by a number-flip animation, optional confetti, and anime-style spark/particle animation.
- Responsive Design: Ensures the application is fully responsive and adapts to different screen sizes (desktop and mobile).
- Sound Effects Toggle: Integrates a toggle for arcade/casino sound effects for button press/reset, with the option to mute.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a casino-like and engaging background.
- Accent color: Teal (#009688) complements the background and is used in reset confirmation button, providing an element of player encouragement.
- Button colors: Blue (#2196F3) for blank shells and Red (#F44336) for active shells, mirroring in-game cues.
- Body and headline font: 'Space Grotesk', a sans-serif for both headers and body text to convey the precision of technology with an impactful feel.
- Simple icons for shell types, reinforcing recognition speed during resets.
- Centered layout with prominent buttons and clear counter displays for intuitive user interaction. Prominent shadow below buttons for floating, 3D vibe.
- Use of smooth animations, such as button pulsing, confetti effects, number flips, deep glassmorphism effect, glowing outer shadows, and subtle bounce/shimmer animations on buttons to enhance user experience and feedback.
